HTML

  • HTML is used to create the structure of a webpage, made up of elements such as text elements, image elements, header elements, and list elements.
  • Common element types used to describe and divide a page are meta-related elements, content division elements, and semantic elements.

    • ELEMENTS

  • head - Contains information about the webpage, or metadata, that helps the browser render the page. Elements within the head are not visible to end users.
  • body - Represents the visible content shown to the user.
  • style - Defines the CSS styles associated with a website.
  • base - Defines the base URL.
  • link - Connects an external resource to the HTML document.
  • meta - Defines metadata such as character set, description, keywords, author, and viewport.
  • h1, h2, h3, etc - Represents the level of heading a text block represents.
  • p - Represents a paragraph.
  • ul, ol, and li - Unordered lists, ordered lists, and list items. Unordered lists are bulleted and order lists are numbered.
  • img - Contains info about images.
  • br - Creates a line of empty space, aka a line break.
  • a - Creates links, stands for anchor.
  • header - Apperas at the top of the page and usually contains a navigation menu or introductory conntent.
  • footer - Appears at the bottom of the page and usually contains info like author, copyright, contact, sitemap, and navigation.
  • section - Defines a section with thematic centent.

    • ATTRIBUTES

  • charset - Sets character encoding.
  • http-equiv - Sets an HTTP header for content.
  • viewport - Sets the default view for our webpage.
  • src - Defines the location of an image.
  • alt - Describes appearance and functionality of an image. Important for screen readers and improving search engine rank
  • href - Points to the URL for links
